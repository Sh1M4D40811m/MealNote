## 🍽️ MealNote
🚧 学習用に個人開発中の食事記録アプリです。
<br>簡単入力で毎日の記録の手間を軽減できるような作りを目指しています。
<br>ユーザーがカスタムでタグを作成でき、タグを元に過去の記録の絞り込み検索ができる機能を入れる予定です。
